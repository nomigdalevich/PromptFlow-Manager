# PromptFlow Manager

React-based prompt management application. Users can add prompts, while an admin can approve, update, or delete prompts. The application keeps track of versions for each prompt and displays detailed information.

## Features
- Add new prompts
- Admin approval and management of prompts
- Update or delete prompts
- Maintain version history for each prompt
- View prompt details and versions

## Technologies
- React
- Redux / @reduxjs/toolkit
- JavaScript (ES6+)
- HTML
- CSS

## How to Run
```bash
# Clone the repository
git clone https://github.com/nomigdalevich/PromptFlow-Manager.git
```

# Navigate to the project folder
cd PromptFlow-Manager

# Install dependencies
npm install

# Start the application
npm start

Open http://localhost:3000 in your browser to view the app
